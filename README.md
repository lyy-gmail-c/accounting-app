# MiniBook · 极简记账

> 一款面向个人的极简记账应用 —— 手机端快速记账，PC 端对账统计，数据本地存储，离线可用。

**在线访问**：[https://lyy-gmail-c.github.io/accounting-app/](https://lyy-gmail-c.github.io/accounting-app/)

---

## ✨ 项目特点

### 📱 双端协作，职责分离
- **手机端**：作为数据采集器，聚焦"快速记一笔"，扁平分类、极简操作，3 步完成记账
- **PC 端**：作为数据管理中心，支持多账户、三级分类、转账记录、对账统计

### 🔒 数据本地存储，隐私安全
- 数据完全保存在浏览器的 **IndexedDB** 中，不上传任何云端服务器
- 无账号体系、无网络同步、无第三方分析 —— **你的账本只属于你**
- 需要同步时：手机导出 JSON → PC 导入，数据流完全在本地掌控

### 🌐 PWA 离线可用
- 首次访问后自动安装到本地，**无需网络也能正常使用**
- 支持添加到主屏幕，启动体验接近原生 App
- Service Worker 缓存策略确保加载迅速

### 🎨 极简设计
- 零外部依赖：**纯 HTML + 原生 JS + CSS**，无任何第三方库
- 启动即用，页面轻量，体积不到 200KB
- 中式财务惯例：涨红跌绿、¥ 人民币符号

## 🚀 快速开始

### 线上使用
直接打开：[https://lyy-gmail-c.github.io/accounting-app/](https://lyy-gmail-c.github.io/accounting-app/)

- 手机用户：推荐用 **Chrome/Safari** 打开后"添加到主屏幕"
- PC 用户：直接在浏览器中访问即可，自动识别为 PC 端界面

### 安装为 PWA（离线使用）
1. 访问上述地址
2. 浏览器地址栏会出现"安装"图标，点击即可
3. 之后即使没有网络也能打开使用

## 📖 功能概览

### 手机端
- 支出 / 收入 快速记账
- 5 大扁平分类（衣食住行 + 其他）
- 今日记录摘要 + 月度统计
- JSON 数据导出（支持系统分享）

### PC 端
- 支出 / 收入 / 转账 三种记账类型
- 多账户管理（微信、支付宝、银行卡、现金...）
- 三级分类体系（共 46 项）
- 账户粒度对账统计 + 分类趋势图
- 数据导入导出 + 智能补齐

## 🛠 技术栈

| 模块 | 技术 |
|------|------|
